# QnA React Application

A simple QnA application to teach React and the ecosystem.

## How to run the App

1. Clone the project
2. Run `npm install`
3. Run `npm start` to run the application
4. Run `npm run db` to run the mock api

## Dependencies

- axios ^0.27
- react ^18
- react-dom ^18
- react-scripts ^5
- sass ^1.5
- json-server ^0.17
